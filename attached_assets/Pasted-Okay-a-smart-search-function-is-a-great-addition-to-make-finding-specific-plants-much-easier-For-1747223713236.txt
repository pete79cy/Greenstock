Okay, a "smart search" function is a great addition to make finding specific plants much easier! For your plant inventory, this means you'll be able to type a query, and the system will look for matches in both the plant names and their scientific names, handling partial matches too.
Here’s how we can implement this:
 * Create a Debounce Hook: This will prevent the system from searching on every single keystroke, waiting instead until you pause typing. This is more efficient.
 * Develop a Backend API Endpoint: This endpoint will take your search query and use it to filter plants from the database. It will search across name and scientificName fields.
 * Integrate Search on the Frontend: We'll add a search input field to your main inventory page, and connect it to the new backend endpoint using the debounce hook.
Let's start by creating the debounce hook.

Step 1: Create the useDebounce Hook
This hook will take a value and a delay, and only return the updated value after the specified delay has passed without further changes.
<lov-write file_path="src/hooks/useDebounce.ts">
import { useState, useEffect } from 'react';
/**
 * Custom hook to debounce a value.
 * @param value The value to debounce.
 * @param delay The debounce delay in milliseconds.
 * @returns The debounced value.
   */
   function useDebounce<T>(value: T, delay: number): T {
   // State and setters for debounced value
   const [debouncedValue, setDebouncedValue] = useState<T>(value);
useEffect(
() => {
// Update debounced value after delay
const handler = setTimeout(() => {
console.log(Debouncing: setting value to ${value} after ${delay}ms);
setDebouncedValue(value);
}, delay);
  // Cancel the timeout if value changes (also on delay change or unmount)
  // This is how we prevent debounced value from updating if value is changed ...
  // .. within the delay period. Timeout gets cleared and restarted.
  return () => {
    clearTimeout(handler);
  };
},
[value, delay] // Only re-call effect if value or delay changes

);
return debouncedValue;
}
export default useDebounce;
</lov-write>
Step 2: Create/Update the API Endpoint for Fetching and Searching Plants
This API route will handle fetching all plants and filtering them based on a search query.
<lov-write file_path="src/pages/api/plants/index.ts">
import type { NextApiRequest, NextApiResponse } from 'next';
import { db } from '@/server/db'; // Adjust path if your db setup is elsewhere
import { plantsTable } from '@/server/db/schema'; // Adjust path to your schema
import { sql, ilike, or, asc } from 'drizzle-orm';
export interface Plant {
id: number;
name: string;
scientificName?: string | null;
plantingYear?: number | null;
quantity: number;
createdAt: Date;
updatedAt: Date;
}
export default async function handler(
req: NextApiRequest,
res: NextApiResponse<Plant[] | { message: string; error?: any }>
) {
if (req.method === 'GET') {
const { search } = req.query;
console.log(API: Received GET /api/plants with search query: '${search}');
try {
  let query = db.select().from(plantsTable).orderBy(asc(plantsTable.name));

  if (search && typeof search === 'string' && search.trim() !== '') {
    const searchTerm = `%${search.trim()}%`;
    console.log(`API: Applying search filter with term: '${searchTerm}'`);
    // Apply search filter to name and scientificName
    // Ensure plantsTable.scientificName is a text type column that can be used with ilike
    // If scientificName can be null, handle it appropriately in your schema or query
    query = query.where(
      or(
        ilike(plantsTable.name, searchTerm),
        plantsTable.scientificName ? ilike(plantsTable.scientificName, searchTerm) : sql`FALSE`
      )
    );
  }
  
  const result = await query;
  console.log(`API: Found ${result.length} plants.`);
  return res.status(200).json(result);

} catch (error: any) {
  console.error('API: Error fetching plants:', error);
  return res.status(500).json({ message: 'Failed to fetch plants', error: error.message });
}

} else {
console.log(API: Method ${req.method} not allowed for /api/plants);
res.setHeader('Allow', ['GET']);
return res.status(405).end(Method ${req.method} Not Allowed);
}
}
</lov-write>
